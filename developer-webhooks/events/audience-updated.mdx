---
title: audince.updated
description: Fired when a contact group (audience) is updated.
---

## When it fires

This event fires when a contact group's name or settings are changed.

## Payload

```json
{
  "event": "audince.updated",
  "data": {
    "uuid": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "name": "xxxx",
    "updated_at": "2025-01-01T12:00:00.000Z"
  }
}
```

## Key fields

| Field | Description |
|-------|-------------|
| `data.uuid` | Group UUID |
| `data.name` | Updated name of the group |
| `data.updated_at` | ISO 8601 update timestamp |
